如何添加新游戏

适合 GitHub Pages 的静态维护流程。

1

部署游戏

先把你的小游戏部署到 GitHub Pages，确认游玩链接可以打开。

2

填写信息

点击“添加游戏”，填入名称、分类、游玩链接、封面图和标签。

3

保存到仓库

临时预览可保存到本地；正式发布请导出或复制 JSON，更新到 games.json 后提交。

添加到游戏库

新游戏信息

  1. 把小游戏部署到 GitHub Pages，复制游玩链接。
  2. 在下面填写信息，点击“添加到本地”先预览。
  3. 确认无误后点击页面右上角导出按钮，把导出的内容替换到 games.json 并提交到 GitHub。
